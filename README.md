Legatora
A Power of Attorney (POA) management system with admin portal.

Features
POA Request Management
Document Verification
Dashboard Analytics
User Management
Tech Stack
Frontend: Next.js, React, Tailwind CSS
Backend: FastAPI, SQLite
Deployment: Vercel (Frontend), Render (Backend)
Getting Started
Prerequisites
Node.js 18+
Python 3.8+
Git
Installation
Clone the repository
Install backend dependencies: cd Backend && pip install -r requirements.txt
Install frontend dependencies: cd Front && npm install
Run backend: cd Backend && uvicorn main:app --reload
Run frontend: cd Front && npm run dev
Deployment
Frontend: Deploy to Vercel
Backend: Deploy to Render
